# ライドシェアアプリ

日本向けのプロトタイプライドシェアアプリケーションです。

## 概要

このアプリは、効率的で安全な移動手段を提供することを目的としています。ユーザーは簡単に乗車をリクエストしたり、ドライバーとして登録したりすることができます。

## 主な機能

- ユーザー登録・認証
- ライドのリクエストと提供
- リアルタイムでのマッチング
- 支払い管理（計画中）

## 技術スタック

- Python 3.x
- Django 5.x
- その他の依存関係は `requirements.txt` を参照
